## Documentation Suggestions

### Summary
This change enhances LSP adapter discovery in the JSON schema store to include "available" LSP adapters (those registered via `register_available_lsp_adapter` but not bound to specific languages). This allows users to see and configure LSP adapters in settings autocomplete even before they're explicitly loaded, improving discoverability of available language servers.
